# Schwarzwälder Hof – Demo Website

Moderne, responsive Restaurant-Webseite (ohne Build-Tools), geeignet für lokales Testen in **XAMPP**.

## Inhalte
- Vorstellung des Restaurants
- Interaktive Speisekarte mit Allergie-Filtern
- Öffnungszeiten inkl. Status „Heute geöffnet?“
- Kontaktinformationen und Reservierungsformular (Demo)

## Lokal testen mit XAMPP
1. Projektordner nach `xampp/htdocs/Schwarzwaelder_Hof` kopieren.
2. Apache in XAMPP starten.
3. Im Browser öffnen: `http://localhost/Schwarzwaelder_Hof/`

## Struktur
- `index.html` – Seitenstruktur
- `styles.css` – responsive Design & Animationen
- `script.js` – Menü-Logik, Filter, Mobile Nav, Reveal-Animation
